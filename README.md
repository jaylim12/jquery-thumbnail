jquery-thumbnail
================

Simple jQuery plugin that creates image thumbnail

Usage
-----
```javascript
$('.thumbnail').thumbnail();
// adjust thumbnail size and the margin that separate the images
$('.thumbnail').thumbnail({ size: 40, margin: 5 });
```

Following is the HTML structure

e.g.:
```html
<div class="thumbnail">
  <img src="img1.jpg" />
  <img src="img1.jpg" />
  <img src="img1.jpg" />
</div>
```
